# CHANGELOG

## [3.0.0] - 2026-05-03

### Додано
- **Web App Manifest**: додано опис застосунку, кольори та іконки.
- **Service Worker**: реалізовано життєвий цикл (install, activate, fetch) та версіонування кешу.
- **Push Notifications**: реалізовано підписку через PushManager та обробку події `push` у SW.
- **Серверний механізм**: додано збереження підписок у `subscriptions.json` та Web Push розсилку.
- **UI**: додано відображення статусу PWA та запит на дозвіл сповіщень.

### Змінено
- Оновлено `app.py` для підтримки VAPID та розсилки сповіщень при підключенні нових клієнтів.
- Оновлено клієнтську частину для реєстрації SW.
